# 🎨 Color & Gradient Tools for Google Sheets
# *Herramientas de Color y Gradientes para Google Sheets*

🏫 This library provides a set of automated tools to help teachers and staff easily color-code, organize, and format spreadsheets. 

*Esta biblioteca proporciona un conjunto de herramientas automatizadas para ayudar a los profesores y al personal a codificar por colores, organizar y dar formato a las hojas de cálculo fácilmente.*

---

## 🇬🇧 How to Use (English)

Once installed, a new menu called **Gradient Tools** 🪄 will appear at the top of your Google Sheets window. To use these tools, simply highlight the cells you want to modify, and select one of the following options from the menu:

* 🌈 **Fill Gradient:** Creates a smooth color transition across your selected cells. 
    * *💡 How it works:* Give the top-left cell a background color, and the bottom-right cell a different background color. Select the entire block, click this option, and the tool will automatically calculate and fill in the gradient between them. (It even works perfectly with merged cells!)
* 📏 **Fill Distinct Linear:** Instantly color-codes a list or range using highly distinct, non-repeating colors. This is perfect for differentiating rows of data or lists of students so that no two adjacent items look the same.
* 🧩 **Fill Distinct Grid:** Similar to the linear option, but mathematically scatters the colors across a 2D grid. This ensures that a cell's immediate neighbors (above, below, left, right) won't have clashing or similar colors.
* ⚙️ **Setup Spreadsheet:** Automatically prepares the current sheet with the standard formatting and structure required for our organization.

---

## 🇪🇸 Cómo usar (Español)

Una vez instalada, aparecerá un nuevo menú llamado **Gradient Tools** 🪄 en la parte superior de tu ventana de Google Sheets. Para usar estas herramientas, simplemente selecciona las celdas que deseas modificar y elige una de las siguientes opciones del menú:

* 🌈 **Fill Gradient (Rellenar Degradado):** Crea una transición de color suave a través de las celdas seleccionadas. 
    * *💡 Cómo funciona:* Ponle un color de fondo a la celda superior izquierda y un color diferente a la celda inferior derecha. Selecciona todo el bloque, haz clic en esta opción y la herramienta calculará y rellenará automáticamente el degradado entre ellas. (¡Incluso funciona perfectamente con celdas combinadas!)
* 📏 **Fill Distinct Linear (Colores Distintivos Lineales):** Codifica por colores instantáneamente una lista o rango usando colores muy distintos que no se repiten. Es ideal para diferenciar filas de datos o listas de estudiantes, asegurando que ningún elemento adyacente se vea igual.
* 🧩 **Fill Distinct Grid (Colores Distintivos en Cuadrícula):** Similar a la opción lineal, pero distribuye matemáticamente los colores en una cuadrícula (2D). Esto asegura que los vecinos inmediatos de una celda (arriba, abajo, izquierda, derecha) no tengan colores similares o que se confundan.
* ⚙️ **Setup Spreadsheet (Configurar Hoja):** Prepara automáticamente la hoja actual con el formato y la estructura estándar requeridos por nuestra organización.
